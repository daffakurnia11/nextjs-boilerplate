# Heading Typography

The `Heading` component is designed to provide a consistent typography hierarchy across your application. This component generally allows the level of heading from `h1` to `h6` for flexible usage in different contexts in the various levels of heading.

## Usage

The `Heading` component can be used to insert text with heading styles into your application. It's crucial for maintaining a semantic and accessible document structure.

```jsx
<Typography.Heading
  level="1"
  as="h1"
  className="text-black"
  styles={{ textAlign: "center" }}
>
  This is an H1 Heading
</Typography.Heading>
```

## Default Styling

By default, the `Heading` component uses the following styles:

```
font-family: "Poppins", sans-serif;
font-weight: 600;
font-size: 2rem; // 32px
line-height: 140%;
```

These styles are applied to match the level of the heading from `1` to `6` with their default level of heading hierarchy.

## How to Use the Component

To use the `Heading` component, simply import it into your component file based on the group of the component, and then use it like any other React component. In this case, the group of `Heading` component is `Typography`. You can specify the `level` prop to determine which heading size you want (from `1` to `6`) and also `as` prop to determine the hierarchy (from `h1` to `h6`):

```jsx
import Typography from "@/components/Typography";

export const MyComponent = () => (
  <div>
    <Typography.Heading level="1" as="h1">
      This is an H1 Heading with large size
    </Typography.Heading>
    // Other content
  </div>
);
```

## Variants

The `Heading` component supports various levels and sizes of headings, which can be specified using the `level` prop. The available variants are `1`, `2`, `3`, `4`, `5`, and `6`.
