# Heading Typography

## Variants

The `Heading` component supports various levels and sizes of headings, which can be specified using the `level` prop. The available variants are `1`, `2`, `3`, `4`, `5`, and `6`.

### Heading 1

This section showcases an example of the `Heading` component at level `1`, along with its default styling. The component utilizes the `.text-5xl` class from **TailwindCSS** for its font size, as specified in `tailwind.config.ts`.

<Canvas of={HeadingStories.Heading1} />

<Source
  code={`
<Typography.Heading as="h1" level={1}>
  Heading 1
</Typography.Heading>
`}
/>
