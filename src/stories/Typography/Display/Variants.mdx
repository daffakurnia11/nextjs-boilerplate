# Display Typography

## Variants

The `Display` component supports various sizes of display headings, which can be specified using the `size` prop. The available variants are `lg`, `base`, and `sm`.

### Display Large

This section showcases an example of the `Display` component at size `lg`, along with its default styling. The component utilizes the `.text-8xl` class from **TailwindCSS** for its font size, as specified in `tailwind.config.ts`.

<Canvas of={DisplayStories.Large} />

<Source
  code={`
<Typography.Display as="h1" size="lg">
  Display Large
</Typography.Display>
`}
/>
