# Small Typography

## Variants

The `Small` component supports various sizes of text, which can be specified using the `size` prop. The available variants are `base` and `sm`.

### Text Small

This section showcases an example of the `Small` component at size `base`, along with its default styling. The component utilizes the `.text-xs` class from **TailwindCSS** for its font size, as specified in `tailwind.config.ts`.

<Canvas of={SmallStories.TextSmall} />

<Source
  code={`
<Typography.Small as="small" size="base">
  Text Small
</Typography.Small>
`}
/>

### Text Extra Small

This section showcases an example of the `Small` component at size `sm`, along with its default styling. The component utilizes the `.text-2xs` class from **TailwindCSS** for its font size, as specified in `tailwind.config.ts`.

<Canvas of={SmallStories.TextExtraSmall} />

<Source
