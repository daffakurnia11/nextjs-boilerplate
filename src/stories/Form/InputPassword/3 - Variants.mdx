# Input Password

## Variants

The `Input.Password` component has some variants that can be modified by own. This component has some properties for providing the design and the requirements of password input component inside the application. Below are the example of some customization of the `Input.Password` component.

### Basic Input

This section showcases the basic component of `Input.Password` that can be used inside the application. The properties that must be used are `name` prop for determining the name of the password input component.

<Canvas of={InputStories.Basic} />
<Source
  code={`
<Input.Password name="password" />
`}
/>

### Adding Label

This section showcases the `Input.Password` component when the component needs to provide the label on the top of the component. Provide the `label` prop for giving the label of the password input component.

<Canvas of={InputStories.AddingLabel} />
<Source
  code={`
<Input.Password name="password" label="Password" />
`}
/>

### Adding Label and Required

This section showcases the `Input.Password` component when the component needs to provide the label and required mark inside the component. Provide the `label` prop for giving the label of the password input component and also `required` to be `true` for adding the `*` mark after the label.

<Canvas of={InputStories.AddingRequired} />
<Source
  code={`
<Input.Password name="password" label="Password" required />
`}
/>

### Adding Placeholder

### Adding Prefix Icon

This section showcases the `Input.Password` component when the component needs to provide the prefix icon inside the component that places on the left of the input text. Provide the `prefix` prop for giving the prefix icon of the input component. This example uses the **React Icons** library for generating the icon.

<Canvas of={InputStories.AddingPrefix} />
