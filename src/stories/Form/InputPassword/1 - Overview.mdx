# Input Passowrd

The `Input.Password` component is designed to provide a consistent password input style across your application. This component requires the `FormGroup` component for handling the value of input component and also has various properties for flexible usage in different contexts or styles of the application.

<Canvas />
<Controls />

## Basic Usage

The `Input.Password` component can be used to create a password input component into your application and wrapped by `FormGroup` component in a various style and configuration. It's crucial for maintaining a sematic and accessible document structure.

<Source
  code={`
<Input.Password name="password" size="base" label="Password" />
`}
/>

## How to Use the Component

The `Input.Password` component is extended from `Input` component, that makes the writing codes are maintainable and clean. The `Input` component is only imported once, without specify which type of input that want to use. You can specify the `size` prop to determine the size of the input component and provide the `name` prop to give the name of the input component. The `Input.Password` component also needs to write inside the `FormGroup` component for creating the form handler function.

<Source code={`
import FormGroup from "@/components/Input/FormGroup";
import Input from "@/components/Input";

export const MyComponent = () => {
  return (
    <FormGroup
      initialValues={initialValues}
      validationSchema={validationSchema}
      onSubmit={onSubmit}
    >
      <Input.Password name="password" label="Password" placeholder="Enter your password" />
    </FormGroup>
  );
}
`} />
