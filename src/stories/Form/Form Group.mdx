# Form Group

The `FormGroup` component is designed to provide a form component that integrated with **Formik** library as a form submission handler and **Yup** library as a validation handler across your application. This component allows the handling of form using validation for flexible usage in different contexts of the application.

<Canvas of={FormGroupStories.Example} />

## Requirements

The `FormGroup` component can be used if **Formik** and **Yup** library has been installed inside the application, so make sure that the libraries have been install using this command below:

```command
npm install formik yup
yarn add formik yup
```

## Basic Usage

The `FormGroup` component basically can be used if the component is provided by the `Input` component. It also requires the initial values and submission function to make the component works.

<Source
  code={`
<FormGroup
  initialValues={initialValues}
  onSubmit={onSubmit}
>
  {/* Input and button components */}
</FormGroup>
`}
/>

## How to Use the Component

The `FormGroup` component can be used to create a form handler in the application with different validation schema and initial value. This component requires the input component from `<Input />` and `<Button />` component as the children or the content of the `FormGroup` component. It's crucial for maintaining a sematic and usable form structure.

```jsx
import * as Yup from "yup";
import FormGroup from "@/components/Input/FormGroup";
import Input from "@/components/Input";
import Button from "@/components/Button";

const initialValues = {
  email: "",
  // Other initial values
};

const validationSchema = Yup.object().shape({
  email: Yup.string().required("Email is required").email("Email is invalid"),
  // Other validation schema
});

const onSubmit = (values: any) => {
  // Logic for handling the submitted data
};

export const MyComponent = () => {
  return (
    <FormGroup
      className="w-80"
      initialValues={initialValues}
      validationSchema={validationSchema}
      onSubmit={onSubmit}
    >
      <Input.Text name="email" label="Email" placeholder="Enter your email" />
      {/* Other input components */}
      <Button.Primary buttonType="submit" fullWidth>Submit</Button.Primary>
    </FormGroup>
  );
}
```
