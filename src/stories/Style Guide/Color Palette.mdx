# Color Palette

The color palette is the foundation for creating a consistent look and feel across the design system. It includes a wide range of colors, such as different shades of **white**, **black**, **primary**, **secondary**, **tertiary**, and **neutral** colors. This carefully selected palette ensures that the user interface looks cohesive. Additionally, it makes it easier to configure the `tailwind.config.ts` file, simplifying the setup process and improving efficiency.
