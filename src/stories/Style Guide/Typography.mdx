# Typography

Typography serves as the foundation for creating a consistent look and feel across the design system. It encompasses a variety of typefaces, font weights, and font sizes carefully selected to enhance readability and visual coherence. Just as a well-curated color palette ensures a cohesive user interface, our typography choices contribute to the overall harmony of the design system. Additionally, our typography settings streamline the configuration process of the `tailwind.config.ts` file, simplifying setup and enhancing efficiency.

export const typography = {
  type: {
    display: '"Merriweather", serif',
    text: '"Poppins", sans-serif',
  },
  weight: {

    extralight: 200,
    light: 300,
    normal: 400,
