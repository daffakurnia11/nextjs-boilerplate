# Primary Button

## Icon Placements

The `Primary` button also supports for placing additional icon in the label of the button. The placement can be added as the prefix icon or suffix icon. This component supports the **React Icons** library for generating the icon and puts into the `prefix` and/or `suffix` properties.

### With Prefix Icon

This section showcases an example of the `Primary` button component with `prefix` prop, along with its icon from **React Icons** library. The placement of the icon is in the front of the label of the button.

<Canvas of={ComponentStories.WithPrefixIcon} />

<Source
  code={`
<Button.Primary size="lg" prefix={<AiOutlinePlus />}>
  Prefix Icon
</Button.Primary>
`}
/>

### With Suffix Icon

This section showcases an example of the `Primary` button component with `suffix` prop, along with its icon from **React Icons** library. The placement of the icon is in the end of the label of the button.

<Canvas of={ComponentStories.WithSuffixIcon} />

<Source
  code={`
<Button.Primary size="lg" suffix={<AiOutlinePlus />}>
  Suffix Icon
</Button.Primary>
`}
/>

### With Prefix and Suffix Icon

This section showcases an example of the `Primary` button component with `prefix` and `suffix` props, along with its icon from **React Icons** library. The placement of the icon is in the front and the end of the label of the button.

<Canvas of={ComponentStories.WithPrefixAndSuffixIcon} />

<Source
  code={`

### Icon in Center

This section showcases an example of the `Primary` button component with `fullWidth` prop to determine the button has a full width size, and `placement` props to configure the placement of the icon. The placement of the icon is in the center which are positioned near the label of the button.

<Canvas of={ComponentStories.IconInCenter} />

<Source
  code={`
<Button.Primary size="lg" fullWidth={true} placement="center" prefix={<AiOutlinePlus />} suffix={<AiOutlinePlus />}>
  Center Icon
</Button.Primary>
`}
/>

### Icon in Between

This section showcases an example of the `Primary` button component with `fullWidth` prop to determine the button has a full width size, and `placement` props to configure the placement of the icon.
