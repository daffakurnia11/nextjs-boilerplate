# Primary Button

## Button Type Variants

The `Primary` button also has some types to ensure it fits the design requirements of the application. The type can be as `default` button that has a background color and border color, `outline` button that only has a border color, and `text` button that does not have a background color and border color.

### Default Button

This section showcases an example of the `Primary` button component at `default` button type that has a background color and border color using `.bg-primary-700` and `.border-primary-700` class from **TailwindCSS**, as specified in `tailwind.config.ts`.

<Canvas of={ComponentStories.ButtonDefault} />

<Source
  code={`
<Button.Primary size="lg" type="default">
  Button Default
</Button.Primary>
`}
/>

### Outline Button

This section showcases an example of the `Primary` button component at `outline` button type that has a transparent background color and border color that using `.bg-transparent` and `.border-primary-700` class from **TailwindCSS**, as specified in `tailwind.config.ts`.

<Canvas of={ComponentStories.ButtonOutline} />

<Source
