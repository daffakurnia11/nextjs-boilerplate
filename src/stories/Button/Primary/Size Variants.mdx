# Primary Button

## Size Variants

The `Primary` button supports different sizes to ensure it fits various contexts and design requirements. Each size variant adjusts padding, maintaining the button's visual balance and clickability.

### Button Large

The large variant has a 10px vertical padding and 24px horizontal padding using `.py-2.5` and `.px-6` class based on **TailwindCSS** class name. The label used the `<Typography.Paragraph>` component with the `base` size.

<Canvas of={PrimaryStories.Large} />

<Source
  code={`
<Button.Primary size="lg">
  Button
</Button.Primary>
`}
/>

### Button Base

The base variant has a 8px vertical padding and 20px horizontal padding using `.py-2` and `.px-5` class based on **TailwindCSS** class name. The label used the `<Typography.Paragraph>` component with the `sm` size.

<Canvas of={PrimaryStories.Base} />

<Source
  code={`
<Button.Primary size="base">
  Button
</Button.Primary>
`}
/>

### Button Small

The small variant has a 6px vertical padding and 12px horizontal padding using `.py-1.5` and `.px-3` class based on **TailwindCSS** class name. The label used the `<Typography.Small>` component with the `base` size.

<Source
