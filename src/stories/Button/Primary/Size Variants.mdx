# Primary Button

## Size Variants

The `Primary` button supports different sizes to ensure it fits various contexts and design requirements. Each size variant adjusts padding, maintaining the button's visual balance and clickability. The available variants are `lg`, `base`, and `sm`.

### Button Large

This section showcases an example of the `Primary` button component at size `lg`, along with its default styling. This variant has a 10px vertical padding and 24px horizontal padding using `.py-2.5` and `.px-6` class based on **TailwindCSS** class name.

<Canvas of={PrimaryStories.Large} />

<Source
  code={`
<Button.Primary size="lg">
  Button
</Button.Primary>
`}
/>

This section showcases an example of the `Primary` button component at size `base`, along with its default styling. This variant has a 8px vertical padding and 20px horizontal padding using `.py-2` and `.px-5` class based on **TailwindCSS** class name.
