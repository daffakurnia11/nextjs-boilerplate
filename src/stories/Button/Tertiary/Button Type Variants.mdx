# Tertiary Button

## Button Type Variants

The `Tertiary` button also has some types to ensure it fits the design requirements of the application. The type can be as `default` button that has a background color and border color, `outline` button that only has a border color, and `text` button that does not have a background color and border color.

### Default Button

This section showcases an example of the `Tertiary` button component at `default` button type that has a background color and border color using `.bg-tertiary-700` and `.border-tertiary-700` class from **TailwindCSS**, as specified in `tailwind.config.ts`.

<Canvas of={ComponentStories.ButtonDefault} />

<Source
  code={`
<Button.Tertiary size="lg" type="default">
  Button Default
</Button.Tertiary>
`}
/>
